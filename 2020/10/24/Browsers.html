В наше время браузеров достаточно много. Для веб-разработчиков есть достаточно большой выбор: Google Chrome, Firefox, Firefox Developer Edition, Vivaldi, Chromium.

Лично я предпочитаю Google Chrome, потому что он достаточно быстрый, умный, хорошо синхронизируется с мобильными устройствами, а также в нём прекрасные инструменты разработчика. Стоит раз попробовать его, как потом сложно будет перейти на какой-то другой браузер.

Вообще, вся тема с браузерами немного преувеличена, так как по сути все они следуют одному стандарту (ну, или стараются ему следовать). Раньше одним из передовых браузеров был Firefox, но так как Google заняла большую часть рынка смартфонов с андроидов, то и пользователей Google Chrome, стало больше, так как Google Chrome поставляется с андроидом с 2012 года.

В 2020 году Mozilla (компания, которая сделала Firefox) уволила 25% процентов сотрудников, поэтому c дальнейшим быстрым развитием Firefox, думаю, стоит подождать. Именно поэтому я и советую перейти на Google Chrome.

Однако, выбор остаётся за каждым и вообще кроме Firefox и Google Chrome есть также и другие браузеры, сейчас я покажу как их установить в Arch Linux:

Установка браузеров в Arch Linux

Сейчас установим несколько пакетов в Arch Linux, а также настроим браузеры

Google Chrome

Устанавливаем yay (Yet another yogurt)

sudo pacman -Sy base-devel git
git clone https://aur.archlinux.org/yay.git
cd yay && makepkg -si && cd -

Затем скачиваем и устанавливаем пакет google-chrome

yay -Sy google-chrome

Firefox

Сразу устанавливаем firefox с помощью pacman

sudo pacman -Sy firefox

Firefox Developer Edition

Firefox Developer Edition - браузер, который специально создан разработчиками для разработчиков. Один из моих фаворитов (после Google Chrome), так как потребляет мало ресурсов и предоставляет достаточно удобные инструменты разработчика.

Устанавливаем так же, с помощью pacman:

sudo pacman -Sy firefox-developer-edition

Vivaldi

Устанавливаем yay (Yet another yogurt) (Не устанавливаем, если уже установлен, естественно)

sudo pacman -Sy base-devel git
git clone https://aur.archlinux.org/yay.git
cd yay && makepkg -si && cd -

Затем скачиваем и устанавливаем пакет google-chrome

yay -Sy vivaldi

Chromium

Сразу устанавливаем Chromium с помощью pacman

sudo pacman -Sy chromium
